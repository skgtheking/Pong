=========================================
Pong Game Setup Guide
=========================================

Welcome to the world of Pong! Follow these instructions to download, compile, and run the Pong game I've created using Unity and C#. Even if you're new to QT, GIT, Unity, or any related tools, this guide will help you navigate through the process smoothly.

=========================================
Prerequisites
=========================================
Before we begin, make sure you have the following installed on your system:
1. Unity Hub: Download and install Unity Hub from [Unity's official website](https://unity3d.com/get-unity/download).
2. Git: If you don't have Git installed, download it from [Git's official website](https://git-scm.com/downloads).
3. Visual Studio or Visual Studio Code: Choose the code editor that you prefer. Download Visual Studio from [here](https://visualstudio.microsoft.com/downloads/) or Visual Studio Code from [here](https://code.visualstudio.com/download).

=========================================
Getting the Source Code
=========================================
1. Open a terminal or command prompt.
2. Navigate to the directory where you want to store the Pong game project.
3. Run the following command to clone the repository:
    ```bash
    git clone https://github.com/your-username/pong-game.git
    ```
   Replace `your-username` with your actual GitHub username.

=========================================
Opening the Project in Unity
=========================================
1. Open Unity Hub.
2. Click on the "Add" button and select the directory where you cloned the Pong game repository.
3. Once added, select the project from the Unity Hub dashboard.

=========================================
Setting Up Unity
=========================================
1. In Unity, open the Scenes folder and double-click on the Pong (1) file to open the game scene.
2. Navigate to File > Build Settings, ensure that the current scene is added to the build settings, and click "Build."

=========================================
Running the Game
=========================================
1. Press the "Play" button in the Unity editor to run the game.
2. Use the following controls:
   - Left Paddle: W (Up), S (Down)
   - Right Paddle: Arrow Up (Up), Arrow Down (Down)

Congratulations! You should now have the Pong game up and running on your machine. Enjoy the classic two-player Pong experience. If you encounter any issues, feel free to explore Unity's documentation or seek help from the Unity community. Happy gaming!
